# ComfyUI共享模型管理器

一个简单易用的可视化工具，帮助您在多个ComfyUI AI软件实例之间共享同一个模型文件夹，节省磁盘空间并简化模型管理。

## ✨ 功能特点

- **📁 模型文件夹共享**：所有ComfyUI实例共享同一个模型文件夹，避免重复下载和存储
- **🎨 可视化界面**：直观的图形界面，0基础小白也能轻松操作
- **➕ 实例管理**：轻松添加、删除和管理多个ComfyUI实例
- **▶️ 一键启停**：点击按钮即可启动或停止ComfyUI实例
- **🌐 快速访问**：一键在浏览器中打开ComfyUI界面
- **💾 配置保存**：自动保存配置，下次启动自动加载

## 📦 安装方法

### 直接使用（推荐）

1. 从GitHub Releases页面下载最新版本的`ComfyUI共享模型管理器.exe`文件
2. 双击运行即可，无需安装Python或其他依赖

### 从源码运行

1. 克隆或下载本仓库
2. 确保已安装Python 3.7+和pip
3. 安装依赖（可选，用于增强界面样式）：
   ```bash
   pip install ttkbootstrap>=1.10.1
   ```
4. 运行主程序：
   ```bash
   python main.py
   ```
